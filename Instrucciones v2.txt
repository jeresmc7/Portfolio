en la terminal de venv (crear una carpeta), el punto va
	django-admin startproject django_portfolio .
ejecutamos el programa con
	python manage.py runserver
con django hay que crear apps de manera paralela
	python manage.py startapp blog
	python manage.py startapp portfolio
Generamos las apps pero hay que indicarle que las creamos
Entonces en django_portfolio/settings.py --> InstalledAPPS 
le agregamos las carpetas que creamos

Cada app tiene su carpeta MODELS, 
la cual tiene clases de python que definden que puedo guardar dentro de esta BD (db.sqlite3)

Para las imagenes hay que instalar Pillow
	pip install Pillow

Por ahi te puede tirar un error de que faltan migraciones de los modelos
	python manage.py makemigrations

Dentro de la carpeta portfolio se crea una carpeta MIGRATIONS, no es recomendable tocarla
Luego hay que iniciar esa migracion
	python manage.py migrate
Ahi genera las tablas

En la url /admin, tenemos un usuario y contrase√±a. Para crearla...
	python manage.py createsuperuser
Se pueden crear usuarios para que utilicen el admin

Luego en el archivo admin.py podemos agregar nuestros modelos

Para rutear las imagenes, en la parte de settings se agrega el MEDIA_ROOT
	MEDIA_ROOT = BASE_DIR / 'media'
